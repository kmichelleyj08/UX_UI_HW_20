Context

Overview

Revitalizing Community Engagement: A Streamlined Website Redesign

The Neighborhood Learning Alliance embarked on a mission to transform their existing website into a more engaging and accessible platform. With a focus on simplifying navigation and enhancing the Events page, the redesign aimed to equip parents with essential information effortlessly. This strategic overhaul was designed to foster greater community involvement and support for the educational programs that shape the next generation.

Problem

Challenge

How can we make the Neighborhood Learning Alliance website more informative and user-friendly for parents?

The existing website struggled with a cluttered interface and insufficient event details, making it difficult for parents to find the information they needed. This challenge necessitated a redesign focused on clarity and ease of use to improve engagement and support for the educational programs.

Design Process

Interviews

To understand the parental needs for the Neighborhood Learning Alliance website, our team interviewed several parents. They highlighted issues like infrequent visits due to a lack of crucial information and difficult navigation . Parents emphasized the need for easily accessible event details and visible contact information.

Many preferred direct communication methods over using the website, indicating it wasn't meeting their needs efficiently. This feedback was crucial for the redesign, pushing for a clearer, well-organized website that effectively supports parents in accessing educational opportunities for their children.

Persona

Stacey Smith's user persona was crafted from interviews with busy parents and used to guide the Neighborhood Learning Alliance website redesign. This profile emphasized the need for streamlined information, centralized program details, and efficient communication, ensuring the website effectively supports engaged parents managing tight schedules.

Definition & Ideation

User Insight

Parents need reliable, accessible information about educational programs to build trust and foster community engagement.

Problem

The current website fails to provide essential information efficiently, limiting parental involvement and community connection.

Value Proposition

We are redesigning the website to be more user-friendly, providing clear and engaging content that encourages community involvement and enhances trust, supported by increases in student enrollment.

Prototyping

Competitor Analysis

Our competitive analysis revealed that direct competitors like Higher Achievement and Boys & Girls Clubs of America excel in easy navigation and strong branding, yet face challenges such as limited programming scope and potentially overwhelming site complexity. Indirect competitors such as The Education Partnership and Khan Academy provide valuable educational resources and extensive subject coverage, but are constrained by regional focus and reliance on user self-motivation.

User flow diagran
Sketches

We created three initial sketches for the Neighborhood Learning Alliance website, each with unique layouts and features. After evaluating these, we merged the best elements into a single, optimized wireframe.

Wireframes

We developed wireframes for the Neighborhood Learning Alliance website to improve user experience with a clear layout. Key features include a streamlined navigation menu, a filterable events page, and dedicated sections for mission statements and testimonials to foster trust and engagement.

Usability Testing

In usability testing, users suggested improvements for the Neighborhood Learning Alliance website: making "upcoming events" clickable to prevent page resets after filtering, adding navigation arrows and a weekend-first calendar layout, linking to personal calendars, and reducing the "events" hero image size for better filter visibility.

Outcome

The final high-fidelity prototype features enhancements like clickable "upcoming events," streamlined calendar navigation, and personal calendar linking. The interface has been optimized to display filtering options immediately by resizing the hero image.
